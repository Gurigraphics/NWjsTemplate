# NWjsTemplate
NW.js Template

### How create build

Install dependences
```js
npm install
```
Build Windows
```js
npm run win
```
Build Linux
```js
npm run linux
```
Build Mac
```js
npm run mac
```

### Production

To production change in json "nwFlavor": "sdk" to "nwFlavor": "normal".

Or add in chromium-args: --disable-devtools
